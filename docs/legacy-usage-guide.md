# 老项目（AsPoem）网站使用说明

本文档面向访客与读者，说明如何在使用老项目（aspoem，当前仓库中的 Next.js 应用）网站时完成浏览、搜索、阅读与按作者/朝代筛选等操作。供新系统复刻功能时对照，也可用于对外说明老站用法。

## 概述

AsPoem 为中文诗词阅读与浏览网站。访客无需登录即可使用主要功能：浏览诗词列表、按朝代或热门/随机查看、全局搜索、阅读诗词详情（含拼音、译文、注解、解析），以及按作者与朝代维度浏览。名句、AI 作诗、游戏等入口当前为「开发中」占位，暂无可用功能。

## 导航与入口

- **Logo**：点击顶部左侧 Logo 进入诗文列表（`/poems`）。
- **顶部菜单**：提供「诗文」「诗人」「名句」「AI 作诗」「游戏」。其中「诗文」为可用入口；「诗人」在菜单上可能显示为不可用，但直接访问 `/authors` 可进入作者列表；「名句」「AI 作诗」「游戏」为占位入口，对应页面为「开发中」。
- **搜索框**：位于页面右上角，点击可打开搜索命令框；也可使用快捷键 **⌘K**（Mac）或 **Ctrl+K**（Windows）打开。输入关键词后显示匹配的诗词，选择某一结果即跳转至该诗的详情页。
- **GitHub 链接**：右上角图标链接至项目仓库，可查看源码或参与贡献。

## 首页

访问根路径（`/`）时，网站会**自动重定向到诗文列表**（`/poems`）。老站没有独立的首页内容，默认展示即为「最近更新」的诗词列表。

## 诗文

### 诗文列表与入口

- **默认列表（最近更新）**：从 Logo 或菜单「诗文」进入 `/poems`，主区域展示最近更新的诗词列表，每页若干条，滚动到底部可点击「加载更多」继续加载。
- **左侧边栏「发现」**：
  - **最近更新**：与默认列表相同，进入 `/poems`。
  - **最受欢迎的**：访问量最高的诗文排行，进入 `/poems/hot`，同样支持列表与「加载更多」。
  - **随机诗文**：进入 `/poems/random`，每次展示随机一首诗；页面上有随机按钮，点击可再随机一首。
- **左侧边栏「朝代」**：按历史顺序列出各朝代，每项显示朝代名称及该朝代诗词数量。点击某一朝代（如「唐 1234」）进入该朝代的诗词列表 `/poems/dynasty/[朝代 slug]`，列表支持「加载更多」。

列表项以卡片形式展示诗题、作者、朝代等信息，点击卡片或标题即进入该诗的详情页。

## 搜索

1. 点击页面右上角搜索框，或按 **⌘K** / **Ctrl+K** 打开搜索命令框。
2. 在输入框中输入关键词（如诗名、作者名、朝代等）。
3. 下方会显示与关键词匹配的诗词列表。
4. 点击其中一条结果，即跳转至该诗的详情页（`/poems/detail/[诗 slug]`）。

搜索为实时查询，输入后会有短暂防抖再请求结果。

## 诗词详情

进入某首诗的详情页（`/poems/detail/[诗 slug]`）后，可进行以下操作：

### 正文与扩展内容

- **标题、作者、朝代**：页面顶部展示诗题、作者（可点击进入作者详情）、朝代等信息。
- **正文与拼音**：主体区域为诗词正文；是否显示拼音由右侧「阅读设置」中的「拼音标注」控制。
- **标签**：若该诗有标签，会在正文下方以按钮形式展示。
- **译文、注解、解析**：页面下方依次展示译文、注解（若有）、解析（赏析）。是否在正文中高亮并展示「原文注解」由阅读设置中的「原文注解」控制。
- **最后更新时间**：页面底部显示该诗内容的最后更新时间。

### 右侧阅读设置

详情页右侧边栏提供「阅读设置」卡片，可调节：

- **简体/繁体**：切换正文用字为简体或繁体（若该开关已实现）。
- **楷体/宋体**：切换正文字体为楷体或宋体。
- **拼音标注**：开关正文上方或行内的拼音显示。
- **原文注解**：开关正文中词汇注解的显示（与译文、解析区域配合）。

设置会随浏览会话保持，便于阅读习惯一致。

### 其他右侧内容

- **纠错与完善/内容贡献**：按钮链接至 GitHub 上该诗对应的 Markdown 文件，可参与纠错或补充内容。
- **作者卡片**：展示当前诗的作者信息及入口，可点击进入作者详情页。

老项目**无**独立「赞赏」页；诗内容中的「解析」为赏析类内容，与「赞赏」不同。若存在打印功能，通常为浏览器自带的打印（如浏览器菜单或快捷键），详情页未单独提供「打印」按钮时，可使用浏览器打印当前页。

## 作者

- **作者列表**：访问 `/authors` 可查看作者列表（标题为「诗人作者」或「某朝代诗人」）。支持分页（如 `?page=1&pageSize=100`）与按朝代筛选（如 `?dynasty=唐`）。列表以表格或卡片形式展示作者名、朝代、作品数等信息。
- **作者详情**：点击某位作者进入 `/authors/detail/[作者 slug]`。页面展示作者姓名、朝代、生卒年（若有）、简介、作品数量，以及**作品列表**（该作者名下的诗词），点击某首作品可进入诗详情页。
- **按朝代作者列表**：从作者列表的朝代筛选进入，或访问 `/authors/dynasty/[朝代 slug]`，可查看该朝代下的作者列表。

## 朝代

- **朝代维度入口**：主要通过**诗文列表页左侧边栏**的「朝代」区域进入。点击某一朝代（如「唐 1234」）进入该朝代的诗词列表 `/poems/dynasty/[朝代 slug]`。
- **按朝代诗词列表**：页面标题为「某朝代 (拼音)」，副标题说明该朝代诗词总数；下方为诗词列表，支持「加载更多」。
- **按朝代作者**：通过作者列表的朝代筛选或 `/authors/dynasty/[朝代 slug]` 查看该朝代的作者。

## 名句 / AI 作诗 / 游戏

顶部菜单中的「名句」「AI 作诗」「游戏」对应路由分别为 `/quotes`、`/ai-generator`、`/games`。当前这些页面为**开发中**占位页，仅显示「开发中」及返回首页等提示，暂无实际功能。菜单上这些项可能显示为不可用（灰色或不可点击）。

## 其他

- **GitHub 链接**：页面右上角提供项目仓库链接，便于查看代码或参与贡献。
- **打印**：若需打印某首诗，可使用浏览器自带的打印功能（如菜单「文件 → 打印」或快捷键），将当前详情页打印为 PDF 或纸质。老站详情页若未提供单独「打印」按钮，即依赖浏览器打印。
- **登录与后台**：老项目具备登录与后台能力（如 `/login`、`/dashboard`），但面向普通访客的「网站使用说明」不涉及登录与后台操作；若需管理员或开发说明，请参考其他文档。
